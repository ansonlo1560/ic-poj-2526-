# 🍼 Smart Baby Cradle (智能感應搖籃車)

這是一個基於 **Arduino Uno** 開發的智能搖籃系統。它能透過高靈敏度麥克風偵測嬰兒哭聲，自動啟動溫柔的原地搖擺模式並播放安撫音樂。同時，支持透過藍牙進行遠端手動控制，並在 OLED 螢幕上即時顯示系統狀態與音量強度。

## ✨ 核心功能
* **哭聲自動感應 (方案 B+C 邏輯)**：
    * **雙重過濾**：結合「長效採樣 (100ms)」與「連續觸發判定」，有效區分短暫雜音（如關門聲、咳嗽）與持續的哭喊聲。
    * **原地搖擺 (Rocking)**：觸發後不會跑遠，而是以輕微的前進後退進行原地安撫。
* **低速平穩控制**：馬達 PWM 經過優化（直行 100, 轉向 75），確保動作溫柔且不會失去控制。
* **可視化狀態欄**：0.96 吋 OLED 即時顯示音量條（Volume Bar）與當前運作模式。
* **多設備 I2C 共享**：MP3 模組與 OLED 螢幕共享 A4/A5 匯流排，系統結構精簡。

## 🛠 硬體清單
| 組件 | 型號/腳位 | 說明 |
| :--- | :--- | :--- |
| **控制器** | Arduino Uno / Nano | 系統大腦 |
| **感測器** | MAX9814 麥克風 (A0) | 負責聲音強度採樣 |
| **顯示屏** | 0.96" OLED (A4/A5) | I2C 位址: `0x3C` |
| **音樂模組** | I2C MP3 模組 (A4/A5) | I2C 位址: `0x7B` |
| **馬達驅動** | L9110S H 橋 | 控制兩組直流馬達 |
| **通訊模組** | HC-05/06 藍牙 (D6, D7) | 接收手機端控制指令 |

## 📐 電路接線說明 (Pin Map)
* **I2C 匯流排 (共享接線)**：
    * `A4 (SDA)` <---> OLED SDA & MP3 SDA
    * `A5 (SCL)` <---> OLED SCL & MP3 SCL
* **馬達控制 (L9110S)**：
    * `D10, D9` <---> 馬達 A
    * `D11, D3` <---> 馬達 B
* **其他**：
    * `A0` <---> MAX9814 Out
    * `D6 (TX), D7 (RX)` <---> 藍牙模組 (交叉接線)

## 🧠 軟體邏輯詳解
### 1. 哭聲判定演算法
系統採用了**連續觸發機制**以減少誤觸：
1.  **採樣**：在 100ms 內抓取最大與最小震幅。
2.  **計數**：若音量超過 `soundThreshold (510)`，計數器 `triggerCount` +1。
3.  **重置**：若兩次大聲間隔超過 2 秒，計數器清零。
4.  **執行**：當計數達到 **3 次**，判定為有效訊號，啟動 6 秒安撫模式。

### 2. 原地搖擺邏輯 (Rocking)
為了安全性，自動模式下不進行長距離移動，而是透過以下循環：
`前進 400ms` -> `停止 400ms` -> `後退 400ms` -> `停止 400ms`

## 📱 藍牙控制指令
使用手機序列埠 App 發送以下指令（不分大小寫）：
* **W / S**：手動前進 / 後退 (速度: 100)
* **A / D**：左轉 / 右轉 (速度: 75)
* **F**：停止運作 / 關閉音樂
* **1 / 2**：切換安撫音樂軌道

## ⚠️ 注意事項
1.  **電力供應**：建議馬達與 Arduino 分開供電，或使用高倍率鋰電池，以避免馬達啟動時的壓降導致 OLED 黑屏或 I2C 當機。
2.  **環境調校**：`soundThreshold` 設為 510 是極限靈敏度，若環境吵雜，建議調低至 450-480。
