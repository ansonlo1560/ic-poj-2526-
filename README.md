# 🍼 Smart Baby Cradle (智能感應安撫搖籃)

這是一個基於 **Arduino Uno** 開發的物聯網（IoT）智能搖籃系統，專為改善嬰兒照護而設計。系統整合了音訊偵測、自動安撫機制與遠端藍牙控制。

## 🌟 核心特色
* **智慧聲音監測演算法 (Algorithm B+C)**：
    * **100ms 深度採樣**：精準捕捉音頻波動，區分環境背景音。
    * **連續觸發邏輯**：需在 2 秒內連續偵測到 3 次異常高音（哭聲）才會觸發，有效避免因關門聲或雜談產生的誤觸。
* **低衝擊原地安撫**：偵測到哭聲後，搖籃會執行「原地搖擺（Rocking）」模式（PWM 100），而非持續移動，確保使用安全性。
* **視覺化狀態監控**：0.96" OLED 實時顯示音量強度條（Volume Bar）與當前操作狀態。
* **I2C 匯流排優化**：透過 I2C 位址管理（0x3C & 0x7B），實現多裝置共用腳位，簡化電路複雜度。

## ⚙️ 硬體腳位分配 (Pin Map)
| 功能 | 組件 | 針腳位置 |
| :--- | :--- | :--- |
| **主控板** | Arduino Uno | N/A |
| **聲音採樣** | MAX9814 麥克風 | **A0** |
| **I2C 數據** | OLED & MP3 模組 | **A4** (SDA) |
| **I2C 時鐘** | OLED & MP3 模組 | **A5** (SCL) |
| **藍牙通訊** | HC-05 (TX/RX) | **D6 / D7** |
| **馬達驅動** | L9110S (A/B) | **D3, D9, D10, D11** |

## 🕹 指令說明
系統支援藍牙序列埠遠端操作：
* `W` / `S`：手動控制前進 / 後退
* `A` / `D`：手動左轉 / 右轉
* `F`：緊急停止 / 靜音
* `1` / `2`：切換安撫音樂軌道

## 🛠 安裝與調試
1.  **函式庫需求**：請確保已安裝 `Adafruit SSD1306`、`Adafruit GFX` 與 `Adafruit BusIO`。
2.  **靈敏度調整**：目前 `soundThreshold` 設為 **510**。若環境較吵，可微調至 **450-480**。
3.  **安全性建議**：建議馬達部分使用獨立電源（如 18650 電池），以避免馬達啟動時的瞬時壓降導致 I2C 通訊中斷。
